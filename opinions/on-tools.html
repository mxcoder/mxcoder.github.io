I love these tools

  1. Linux terminal and Bash: pretty much automates anything.
  2. jq: to work with JSON in the CLI
  3. DuckDB: to work with small/medium data pretty much anywhere.
  4. Benthos: now Redpanda Connect, to move data pretty much from and to anywhere.
  5. GCP BigQuery and BigQueryML: exquisite architecture, astounding performance, probably too expensive for small companies.
  6. Python 3: it simply does the work.
